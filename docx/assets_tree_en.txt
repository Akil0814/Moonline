assets/
|  Root resource directory for the project, containing runtime images, audio, fonts, and configuration files.
|-- audio/
|  Stores audio resources, mainly character voices, sound effects, and potentially background audio in the future.
|   `-- character/
|      Organizes voice lines and character-related audio by character.
|       `-- RyougiShiki/
|          Stores voice lines and audio assets used by the character Ryougi Shiki.
|-- config/
|  Stores JSON configuration files used to describe characters, enemies, scenes, and global system settings.
|   |-- character/
|   |  Stores the character list and per-character configuration for animation, attacks, audio, skills, and related data.
|   |-- enemy/
|   |  Stores the enemy list and configuration for enemy attributes, animations, and behavior-related data.
|   |-- global/
|   |  Stores global settings such as input, audio, UI layout, and system-level parameters.
|   `-- scene/
|      Stores level or scene configuration, such as the stage list, scene data, and enemy placement.
|-- fonts/
|  Stores font files used by the game.
|   `-- IPix.ttf
|      A font resource currently used by the project.
|-- preload/
|  Stores resources loaded early for the preload screen, such as loading images, icons, and transition assets.
|   `-- icon/
|      Stores icons and icon frame sequences used during startup or loading.
`-- textures/
   Stores texture and image resources, which form the main visual asset directory of the project.
    |-- character/
    |  Organizes portraits, animation frames, attack effects, and other character-related images by character.
    |   `-- RyougiShiki/
    |      Stores animation frames, skill images, and character art for Ryougi Shiki.
    |-- enemy/
    |  Stores textures, animation frames, and related visual resources for enemies.
    |-- global_effect/
    |  Stores reusable global effect textures, such as shared hit effects, flashes, or system effects.
    |-- stage/
    |  Stores textures related to scenes, maps, or stage backgrounds.
    `-- ui/
       Stores interface images such as buttons, panels, icons, and other UI textures.
